# ⚔️ Medieval RPG Bot

یک ربات رول‌پلی قرون وسطایی برای تلگرام با سیستم‌های پیچیده و رنکینگ جهانی.

## 🎮 ویژگی‌های اصلی

- **سیستم کاراکتر**: ایجاد و ارتقاء پروفایل
- **جنگ**: مبارزات ترن‌بیس با بازیکنان و باس‌ها
- **ارتش**: انتخاب و ساخت ارتش شخصی
- **اقتصاد**: کسب پول، خزانه، و انتقال پول
- **قلمرو**: کسب و مدیریت قلمرو و مالیات
- **کوئست**: مأموریت‌های روزانه و دانجن‌ها
- **رنکینگ**: بر اساس ثروت، قدرت، لول، و قلمرو
- **پنل ادمین**: کنترل و مدیریت بازی

## 📊 حداکثر لول: 100

## 🔧 فناوری

- **Python 3.9+**
- **Aiogram** (تلگرام API)
- **PostgreSQL** (دیتابیس)
- **Railway** (میزبانی)

## 📦 نصب و راه‌اندازی

```bash
git clone https://github.com/ARTIN14301/medieval-bot.git
cd medieval-bot
pip install -r requirements.txt
```

## 🚀 شروع

```bash
python main.py
```
