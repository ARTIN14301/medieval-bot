# Medieval RPG Bot 🏰⚔️

یک ربات تلگرام با موضوع قرون وسطایی با سیستم RPG کامل، شامل جنگ، سطح‌بندی، قلمرو و درجه‌بندی!

## ✨ ویژگی‌های اصلی

### 🎮 سیستم بازی
- **سطح‌بندی**: تا سطح 100 با سیستم تجربه پویا
- **پول و اقتصاد**: سیستم طلا با محدودیت‌های بازی‌شناختی
- **ارتش و سربازان**: 6 نوع سرباز با ویژگی‌های مختلف
- **جنگ**: جنگ تک‌نفره، PVP و جنگ با بوس
- **کوئست**: کوئست روزانه با پاداش محدود
- **قلمرو**: فتح و مدیریت قلمرو برای درآمد
- **دانجن**: دانجن‌های چند سطحی با مختلف‌ترین سختی‌ها

### 📊 سیستم رتبه‌بندی
- رتبه‌بندی ثروت (طلا)
- رتبه‌بندی قدرت (حمله + دفاع)
- رتبه‌بندی سطح
- رتبه‌بندی قلمرو

### 👑 سیستم مدیریت
- پنل ادمین
- مدیریت کاربران
- کنترل اقتصاد

## 🛠️ فناوری‌های استفاده‌شده

- **Aiogram 3**: کتابخانه تلگرام برای Python
- **SQLAlchemy**: ORM برای مدیریت دیتابیس
- **PostgreSQL**: دیتابیس (محلی: SQLite)
- **Python 3.11+**: زبان برنامه‌نویسی

## 📦 نصب و راه‌اندازی

### الزامات
- Python 3.11 یا بالاتر
- pip
- PostgreSQL (برای تولید)

### نصب محلی

```bash
# کلون کردن مخزن
git clone https://github.com/ARTIN14301/medieval-bot.git
cd medieval-bot

# ایجاد محیط مجازی
python -m venv venv
source venv/bin/activate  # در Windows: venv\Scripts\activate

# نصب وابستگی‌ها
pip install -r requirements.txt

# تنظیم متغیرهای محیط
cp .env.example .env
# سپس .env را ویرایش کنید و توکن بات را اضافه کنید

# اجرای بات
python main.py
```

برای استقرار روی Railway، [RAILWAY_SETUP.md](RAILWAY_SETUP.md) را ببینید.

## 📖 دستورات

### حساب کاربری
- `/start` - شروع بازی
- `/register` - ثبت‌نام کاربر جدید
- `/profile` - نمایش پروفایل

### جنگ و معارکه
- `/solo` - جنگ تک‌نفره
- `/pvp [user_id]` - جنگ با بازیکن دیگر
- `/boss` - جنگ با بوس
- `/daily` - کوئست روزانه

### فروشگاه و موجودی
- `/shop` - فروشگاه
- `/shop_units` - خریدن سربازان
- `/shop_items` - خریدن آیتم‌ها
- `/inventory` - نمایش موجودی

### ارتش و قلمرو
- `/army` - مدیریت ارتش
- `/territory` - مدیریت قلمرو

### سیستم
- `/dungeon` - دانجن‌ها
- `/ranking` - رتبه‌بندی جهانی
- `/help` - راهنما

### پنل ادمین
- `/admin` - دسترسی ادمین
- `/admin_stats` - آمار بازی
- `/admin_ban [user_id]` - بن کردن کاربر
- `/admin_give [user_id] [amount]` - دادن طلا
- `/admin_setlevel [user_id] [level]` - تنظیم سطح
